# 🚀 限界はあなたの頭の中にしかない 学習アプリ

ジェイ・エイブラハム『限界はあなたの頭の中にしかない — 小さなアクションで、最大の成果を引き寄せる』を素材にした、ありえる楽考スタイルの学習アプリ。

**公開URL**: <https://arieru39.github.io/ariel-apps/genkai/>

## この本の主張（1文）

**「人間の限界は自己が課した心的構築物であり、判断軸を内側に持てば、小さなアクションが複利で品格ある人生を作る」**

エイブラハム自身が18歳でホームレス／学習障害から出発し、貢献志向のマインドセット × 経験からの学習 × クリティカル思考の積み重ねで、ビジネスを「道」として歩む。取引ではなく貢献、ルールではなく判断軸を目指す。

## 11テーマ × 3層構造

- **第1層 · 心層** — `#limit_illusion` / `#philosophy_vs_tactic` / `#failure_resilience` / `#value_creation_contribution`
- **第2層 · 実践層** — `#small_action_compounding` / `#extracting_from_experience` / `#socratic_dialogue` / `#mentor_learning_uniqueness` / `#critical_thinking_leverage` / `#customer_psychology_ethics`
- **第3層 · 品格層** — `#character_elegance`

層は順番に上るのではなく、螺旋的に往復します。心が変わる→実践が変わる→経験が積み重なる→さらに深い心の在り方が生まれる→やがて品格として結晶する。

## 中核ループ

```
[選ぶ]     今日触る #LO を1つ選ぶ
   ↓
[入る]     本文抜き書き ＋ 種の問い ＋ 実務例プロンプト
   ↓
[書く]     自分の言葉で100〜300字
   ↓
[写す]     ルーブリック4段階に自己マーク＋根拠1行
   ↓
[つなぐ]   たまかつ接続： 何のために？ 誰のために？
   ↓
[晒す]     場に投げる（Markdown書き出し／保存）
```
