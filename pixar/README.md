# ピクサー流創造する力 — 学習アプリ

エド・キャットムル『ピクサー流創造する力 ── 真実と勇気の文化をつくる』（ダイヤモンド社／原題 *Creativity, Inc.*）を教材にした、ありえる楽考スタイルの学習アプリ。

## 本の中核主張（1文）

> **問題は必ず起こる。問題の多くは隠れて見えない。マネジャーの仕事は、率直さ・失敗・偶発性・目に見えないものと向き合う場を守り続けることで、創造性を個人技を超えた組織文化に沈殿させることだ。**

## 構成：11テーマ × 3層構造

### 第1層 · 見る目 — 気づかない構造を見る
- `seeing_the_hidden_structure` — 気づかない構造を見る（第1章 会議テーブルの話）
- `the_hidden_problem` — 隠れた問題を見る（第9章 カッサンドラの寓話）
- `broaden_the_view` — 視野を広げる（第10章 パンクしたキャンピングカー）

### 第2層 · 守る術 — 新しいものを守る
- `story_is_king` — 物語が一番偉い（第4章 ピクサーの2大原則）
- `braintrust` — ブレイントラスト（第5章 率直な意見の場）
- `fear_and_failure` — 失敗と恐怖心（第6章 早く安く失敗する文化）
- `ugly_baby_and_beast` — 醜い赤ん坊と貪欲な野獣（第7章 生まれたてを守る）

### 第3層 · 続ける仕組み — 持続する文化を回す
- `manager_removes_obstacles` — 障害物を取り除く（序章 リーダーの真の使命）
- `change_and_randomness` — 変化と偶発性（第8章 変わらないと言った直後に）
- `unmade_future` — 未来はまだ存在しない（第11章 創造性はマラソン）
- `notes_day` — ノーツ・デイ（第13章 全員で文化を更新する）

## 中核ループ

```
率直に話せる場を守る（Braintrust）
    ↓
失敗が早く表に出る（Fear and Failure）
    ↓
見えなかった問題が見える（The Hidden）
    ↓
次のマラソンを走れる（Unmade Future）
    ↓
（文化そのものを見直す：Notes Day）
    ↓
最初に戻る
```

この4つ（+1）が噛み合い続けるとき、創造性は個人技ではなく組織の文化として立ち上がる。

## 各要素

- **各LOに 4段階ルーブリック**（Fast/Slow 位相付き）
- **各LOに 観点クイズ**（4択・5層フィードバック：stance / phase / phase_note / author_view / slow_question）
- **8問の関係性クイズ**（テーマ横断・組織の見えない問題を映す鏡）
- **第1層ガイドバナー**（初めての方向け導線）
- **エンジンノート**（Deliberate Practice 統一原理）
