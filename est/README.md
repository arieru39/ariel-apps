# est 学習アプリ

Werner Erhard の est コースの視座を、**3層 × 10LO × 4段階ルーブリック** に落とし込んだ学習アプリ。

## この本を1行で

> **「私」の物語をいったん脇に置いて、"今この場に何が現れているか" から生き直す。**
> Doing（何をするか）ではなく Being（誰であるか）から始めるための、体験のリセット装置。

## 3層 × 10LO

- **第1層 · 認知の解体**（「私は正しい」を手放す）
  1. `self_illusion` — 自我の錯覚
  2. `right_wrong_trap` — 正しさの罠
  3. `assumption_dissolution` — 前提の溶解

- **第2層 · 体験の転換**（Doing の書き換え）
  4. `seriousness_sickness` — 深刻さという病
  5. `success_failure_reversal` — 成功と失敗の反転
  6. `chance_choice` — 偶然と選択
  7. `experiencing_now` — 今を体験する

- **第3層 · 生き方の統合**（Being への転回）
  8. `being_over_doing` — Being の優位
  9. `multiple_selves` — 複数の自己
  10. `controlled_folly` — コントロールされた愚行

## 中核ループ

1. 関係性クイズ（8問）で全体を掴む
2. 各LOで：観点クイズ → 動画 → 問いに書く → ルーブリックで自己評価
3. Slow の反復で Fast に沈殿していく（Ericsson の Deliberate Practice）
