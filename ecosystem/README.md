# 価値共創コミュニティのエコシステムづくり 学習アプリ — 場を設計する側の視点版

『価値共創コミュニティで「たまかつ」を生きる』鈴木利和 著（ありえる楽考 Kindle出版）を素材に、**「個人が場で生きる」ではなく「場そのものを設計・構築する」側の視点で読み直す** ありえる楽考スタイルの学習アプリ。

## 姉妹アプリとの関係

同じ本『たまかつ』を素材に、2つの視点で読める姉妹アプリを提供しています：

| アプリ | 視点 | 中心マイルストーン |
|--------|------|------------------|
| **[[たまかつ]]**（第3プロダクト） | 個人が場で熟達し生きる側 | 個人の生き方の羅針盤 |
| **[[エコシステムづくり]]**（第6プロダクト・本アプリ） | 場を設計・構築する側 | みんなでつくる場（波紋のプログレッション） |

同じ章群を裏表で読むことで、「場で生きる自分」と「場をつくる自分」の両方の視座が獲得できます。

## この教材の3つの編集軸

### 1. 「個人の実践 → コミュニティ設計」への翻訳

たまかつ本の各章を、「私はこう生きる」ではなく「どんな場を私は設計するか」に読み替える。各テーマに `【個人版】→【コミュニティ版】` の1行変換を明記。

- 【個人版】自分が学び熟達するための場を持つ → 【コミュニティ版】インプット→場→アウトプットが循環する生体系エコシステムを設計する
- 【個人版】自分が卓越した個人になる → 【コミュニティ版】凡人が集合天才として動く場を仕掛ける
- 【個人版】自分がリーダーとして引っ張る → 【コミュニティ版】答えを持たず山の頂上だけ指さすガイド役を担う

### 2. 「1人の決意 → 波紋 → エコシステム → 世界の場」というプログレッションをイメージする

中心マイルストーンは **🌱 みんなでつくる場**。1人の決意が波紋を起こし、同心円状に広がってエコシステムになり、やがて世界の中に新しい場をつくる。この4フェーズが、あなたの場を設計するときの"羅針盤"になります。**「今、私の場はどのフェーズにいるか」の見立てが、次の一手の起点。**

### 3. 「コンセプト→モデル→ストーリー」の3層構造

- **📐 コンセプト** — なぜエコシステムが必要か
  - 価値共創コミュニティの青写真／集合天才／ファシリテーターの立ち位置
- **🧩 モデル** — どう場を編むか（型）
  - 三層構造（Why/What/How）／楽友の3原則／みんなでつくる場／グループコーチング4ステップ／プロダクトをつくる／チーム解散の力
- **🎬 ストーリー** — どう時間をかけて波及するか
  - ビジョンでリクルーティング／Kindle出版する

## 11テーマ × 3層構造

### 第1層 · 基盤 — なぜエコシステムなのか（concept）
1. **📐 価値共創コミュニティの青写真** — インプット→場→アウトプットの循環
2. **📐 集合天才：凡人の集まりが非凡を生む** — 1人のカリスマに依存しない設計
3. **📐 ファシリテーターの立ち位置：教えるのではなく導く** — 先生ではなくガイド

### 第2層 · 実装 — 対話と三層構造で場を編む（model）
4. **🧩 三層構造：Why / What / How を分けて考える** — オーナー／ファシリ／参加者
5. **🧩 楽友の3原則：対等・率直・肯定** — 三層を機能させる横のフラットな関係
6. **🌱🧩 みんなでつくる場：1人の決意から始まるエコシステム** — 中心マイルストーン
7. **🧩 グループコーチング4ステップ：エコシステムのエンジン** — 振り返り→発表→傾聴→共有

### 第3層 · 波及 — 場を育て、社会に還す（story含む）
8. **🧩 プロダクトをつくる** — 場から生まれた実践を、誰かの『ありえない』を『ありえる』に変えるプロダクトへ
9. **🎬 ビジョンでリクルーティング** — お願いから招待状へ
10. **🧩 チーム解散の力** — クラス替えでコンフォートゾーンを出る
11. **🎬 Kindle出版する** — 30点で世に出す軽やかさ

## 中核ループ

各テーマは以下の5ステップで学習：

1. **個人→コミュニティ設計の変換を読む** — 「【個人版】→【コミュニティ版】」の1行で、視点を場のデザイン側に合わせる
2. **抜き書きを読む＋動画を観る** — 原著の該当箇所（重要箇所は **太字**）＋章別YouTube解説（各テーマに紐付け済）
3. **実務例を書く** — 自分の場のデザインに置き換えて言語化
4. **自己評価する** — 4段階ルーブリックで現在地を確認
5. **他アプリで深める** — 関連する3〜5つの他アプリのテーマへリンク

## クロスリンク（5アプリ・エコシステム）

各テーマから、以下の4アプリの関連テーマにリンク：

- **[[たまかつ]]**：同じ本の『個人が場で生きる』側（表裏の関係）
- **[[みんなが「話せる」学校]]**：対話による場づくり（学校を『エコシステム』として設計）
- **[[コンテキスト・リーダーシップ]]**：山口周のリーダーシップ論
- **[[限界はあなたの頭の中にしかない]]**：ジェイ・エイブラハムの思考法
- **[[Good to Great 個人キャリア版]]**：ジム・コリンズの経営研究

中心マイルストーン『みんなでつくる場』は他アプリを含めて **計 5 件** の関連リンクを設置。
