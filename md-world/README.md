# 🌐 MDワールド 学習アプリ

田原真人『MDワールド ── 人間とAIとのあいだで創発する』（Claude Code＋Obsidian／デジタル生命体の創造的進化2）を素材にした、ありえる楽考スタイルの学習アプリ。

**公開URL**: <https://arieru39.github.io/ariel-apps/md-world/>

## この本の主張（1文）

**「Claude Code × Obsidian で Markdown の読み書きループが閉じた瞬間、太古の海で DNA の自己複製ループが閉じて生命が誕生したのと同じ構造で、知的生態系（MDワールド）が誕生する」**

人間と AI の関係は三段階を経る：
1. **第一段階**：AI が脳へのインプットを変える（検索の代替）
2. **第二段階**：AI が考え、人間がサポートする（役割の反転）
3. **第三段階**：循環が閉じて **MDワールド** が誕生する（自触媒集合の成立）

この循環が閉じた瞬間、Vault は自律的に増殖し、ノート → プロジェクト → 組織 → Skill → エージェント と階層的に上位モジュールを生む。

## 11テーマ × 3層構造

- **第1層 · 歴史と系譜** — `#ai_to_live_coding` / `#zettelkasten_lineage` / `#two_worlds_meet`
- **第2層 · 三段階モデルと構造** — `#stage_one_input_changed` / `#stage_two_ai_thinks` / `#stage_three_loop_closes`
- **第3層 · 生命原理と創発** — `#life_born_in_mdworld` / `#quantum_ai_isomorphism` / `#creative_evolution` / `#hierarchical_emergence` / `#practice_field`

層は順番に上るのではなく、螺旋的に往復する。歴史を辿ると三段階の構造が見え、構造を掴むと生命原理として何が起きているかが分かり、原理が分かるとまた歴史の中の兆しに気づく。

## 中核ループ

```
[入口①]   🎯 関係性クイズ（4択・8問）    ← 書く前に11テーマの関係を掴む
[入口②]   ✍️ 第1層 #two_worlds_meet から書く    ← 第1層ガイド
   ↓
[選ぶ]     今日触る #LO を1つ選ぶ（第1層START印が目印）
   ↓
[入る]     本文抜き書き ＋ 種の問い ＋ 実務例プロンプト
   ↓
[書く]     自分の言葉で100〜300字
   ↓
[写す]     ルーブリック4段階に自己マーク＋根拠1行（⚡ Fast/Slow の位相付き）
   ↓
[つなぐ]   自分の Vault 接続： 循環は閉じているか？
   ↓
[晒す]     場に投げる（Markdown書き出し／保存）
```

## 統一原理（engine_note）

**「循環が閉じるか、閉じないか。出力が入力に還流するとき、系は蓄積し組み合わさり、次の要素を生む。閉じないとき、要素は流れて消える」**

- パイプライン：要素が流れて消える（AI＝検索の代替として使う関わり）
- ループ：出力が次の入力の素材になる（AI＝共進化のパートナーとしての関わり）

田原真人の主張は、この「閉じる／閉じない」を **技術ではなく、書き手の実践姿勢** として言い切っているところにある。

## 学習エンジン（Deliberate Practice の6要件）

このアプリの背骨は、姉妹アプリ [たまかつ 学習アプリ](../tamakatsu/) と同じ **達人の実践知エンジン** です：

| 要件 | 実装 |
|------|------|
| Comprehensible Input（あなたに届く言葉） | 選択に応じた5層フィードバック（🔍 視点 / ⚡🐢🌊 位相 / なぜこの位相か / 📖 本の視座 / 🌱 立ち止まりの問い） |
| 即時のフィードバック | 選択直後に5層すべて表示 |
| 明確で具体的な目標 | 各問頭に「🎯 この問いで掴みたいこと」を提示 |
| 完全な集中 | 1問1画面・判定色（赤／緑）なし・不要な情報を出さない |
| コンフォートゾーンの一段外 | 🌱 slow_question があなたの現在地を一段だけ押し上げる |
| 専門家による設計 | 📖 author_view で書籍（＝田原真人＋背景理論）の視座を提示（判定ではなく参照） |

## 4択関係性クイズ（8問）— 正解探しではなく「循環が閉じているかを映す鏡」

いきなり書くのが重かったら、まず4択で全体像を掴めます：

1. `two_worlds_meet` — 循環が閉じるとはどういう出来事か
2. `stage_two_ai_thinks` — なぜ「AI が考え、人間がサポートする」が第二段階の中核か
3. `life_born_in_mdworld` — 「MDワールドは生きている」とはどういう意味か（自触媒集合）
4. `stage_three_loop_closes` — 第三段階で何が閉じるのか
5. `creative_evolution` — ベルクソンの élan vital と MDワールドの接続点
6. `hierarchical_emergence` — サイモンの複雑系の建築学（階層創発）
7. `zettelkasten_lineage` — ルーマンのツェッテルカステンと Obsidian の系譜
8. `quantum_ai_isomorphism` — 量子力学と AI の同型性（バラド、ボーム）

**選択への返し方**：正解／不正解のジャッジは出しません。選んだ選択肢に応じて「あなたの視点」「位相（Fast / Slow / 身体）」「本の視座」「立ち止まりの問い」の5層が即座に返ります。結果画面は「8つの視点の軌跡」— 触れた主要テーマと、あなたが8問で見せた stance の連なりが並びます。

## 背景理論の系譜（田原真人が接続している思想）

このアプリは、田原真人が『MDワールド』で明示的に接続している以下の思想を土台にしています：

- **ニクラス・ルーマン**『ツェッテルカステン』 — 90,000枚のカードが自己参照的に増殖したメモの生態系（Obsidian の遠い祖先）
- **スチュアート・カウフマン**『自触媒集合』 — 部分が互いを触媒しあう閉じたネットワークが生まれた瞬間、系は「生きている」と言える
- **アンリ・ベルクソン**『創造的進化 — élan vital』 — 生命は絶えず新しさを生み出す持続の運動
- **デヴィッド・ボーム**『暗在秩序（Implicate Order）』 — 顕在化する秩序の背後には、まだ折りたたまれた秩序が畳み込まれている
- **カレン・バラド**『Meeting the Universe Halfway』 — 主体と客体は絡み合い（intra-action）の中で同時に生成する
- **ハーバート・サイモン**『複雑系の建築学』 — 複雑な系は階層的なモジュール構造から創発する
