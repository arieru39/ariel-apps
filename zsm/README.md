# 全社戦略メモ ジェネレーター（zsm）

戸田さん戦略スクール『パートナー型戦略コンサルタント養成講座』のフレーム（Day1〜Day4）に沿って、**全社戦略メモを1枚の設計図として量産する**ためのウィザード型アプリ。

## 何ができるか

- **11ステップのウィザード**で、順番に問いに答えていくだけで戦略メモが完成する
- **各ステップで戸田流フレームの解説**を表示（ピラミッド・ストラクチャー／3C／松竹梅／戦略ピラミッドなど）
- **ありえる楽考の記入例**を「クリックで開閉」できる形で埋め込み（真似ぶための実例）
- **ライブプレビュー**で、書きながらリアルタイムに完成形のMarkdownが見える
- **localStorage自動保存**で、ブラウザを閉じても書きかけが消えない
- **1クリックでMarkdownコピー／`.md`ダウンロード**（Obsidianにそのまま貼り付け可能）

## 11ステップの構成

| # | 章 | 使うフレーム |
|---|---|---|
| 0 | エグゼクティブ・サマリー | ピラミッド・ストラクチャー |
| 1 | 目的・目標 | 目的（Why）と目標（What）の分離 |
| 2 | 現状分析 | 3C（Customer / Competitor / Company） |
| 3 | 戦略の方針（Where / How to Win） | 松竹梅モデル／必要条件・十分条件／バリューチェーン |
| 4 | 戦略のピラミッド | ヒト・モノ・カネ |
| 5 | やらないことの明示 | Where NOT to Play |
| 6 | 90日アクション | 30日／60日／90日の3段階 |
| 7 | 想定リスクと対処 | 問題発見の5視点 |
| 8 | 対話のための問い | 対話型戦略（答えはマーケットにしかない） |
| A | 依拠フレーム一覧（付録） | トレーサビリティ確保 |
| B | 独自資産（付録） | 自社IP／概念／道具立て |

## 使い方

1. `https://arieru39.github.io/ariel-apps/zsm/` を開く
2. 上部「基本情報」に会社名・作成者・作成日を入力
3. 各ステップの問いに答えていく（分からない項目は空欄OK・後で戻れる）
4. 右パネルの「ライブプレビュー」で完成形を確認
5. `📋 Markdownをコピー` または `💾 .mdをダウンロード` でObsidianに持ち込む

## 設計方針

- **ウィザード型**：11章を一度に見せると圧倒される。1ステップずつ集中して書けるように分割
- **フレーム解説とありえる楽考例を隣接**：戸田流の理論と、ありえる楽考の実践を同じ画面で対比できる
- **プレビュー常時表示**：書き終わってから初めて完成形を見るのではなく、書きながら全体像を掴む
- **絞り込みを促す**：章5「やらないこと」を必ず通る構成にし、"戦略のようで戦略でないもの"化を防ぐ
