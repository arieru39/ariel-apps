# みんなが「話せる」学校 学習アプリ — 対話による場づくり版

『みんなが「話せる」学校 — 対話で学びと挑戦の土壌を創る』吉村春美 著（学事出版・2024）を素材に、**企業のグループコーチング（GC）と事例共有会（TTP）の "学校版" として読み直す** ありえる楽考スタイルの学習アプリ。

## この教材の3つの編集軸

### 1. 学校の実践を、企業の GC + TTP に翻訳する

本書は学校の教員・校長・ミドルリーダー向けに書かれている実践書。ですが、そこで扱われているのは **「対話による適応課題の解決」** という、企業のGC+TTPで扱おうとしていることそのもの。各テーマに「【学校版】→【企業版 GC + TTP】」の変換を明記。

- 【学校版】適応課題を見極める → 【企業版】GC/TTPで扱うべきは業務手順ではなく判断基準・価値観のズレ
- 【学校版】対話の4モード → 【企業版】GCの中で儀礼→討論→内省→生成のシフトをファシリが起こせるか
- 【学校版】職員会議を「話せる」場に → 【企業版】伝達型定例に問いを仕込む

### 2. 対話による場づくりを「イメージする」アプリ

中心マイルストーンは **🌱 対話の4モード（儀礼 → 討論 → 内省 → 生成）**。アダム・カヘンの分類が、「あなたが作りたい対話の場のグラデーション」そのものになっています。**「今この場は何モードにいるか」の見立てが、場のデザインの起点。**

### 3. 「コンセプト→モデル→ストーリー」の3層構造

- **📐 コンセプト** — なぜ対話が必要か
  - 適応課題を見極める／心理的安全性／自分の想定を保留する
- **🧩 モデル** — どう場を編むか（型）
  - NVC 4ステップ／対話の4モード／生成的な対話／内省的に聴く／職員会議の再設計／促進的リーダーシップ／ミドルリーダー
- **🎬 ストーリー** — どう時間をかけて波及するか
  - 保護者・地域とのエコシステム共創

## 11テーマ × 3層構造

### 第1層 · 土台 — 場を見立てる（concept）
1. **📐 適応課題を見極める** — 技術的問題と適応課題の区別
2. **📐 心理的安全性を組織の土壌に** — 4つの不安（無知・無能・邪魔・ネガティブ）
3. **📐 自分の想定を保留する** — メタ認知
4. **🧩 NVC：観察・感情・願い・要求** — 共感的コミュニケーションの型

### 第2層 · 実装 — 対話で場を編む（model）
5. **🌱🧩 対話の4モード** — 儀礼→討論→内省→生成（中心マイルストーン）
6. **🧩 生成的な対話** — 立場を超えて話す
7. **🧩 内省的に聴く・共感的に聴く** — 深く聴く
8. **🧩 職員会議を「話せる」場に再設計** — 伝達型定例の中に対話を仕込む

### 第3層 · 波及 — エコシステムとして育てる（story含む）
9. **🧩 促進的リーダーシップ** — 校長のファシリテーター化
10. **🧩 ミドルリーダーの実践** — 権限より影響力で動く層
11. **🎬 保護者・地域との共創** — 半年〜数年のプログレッション

## 中核ループ

各テーマは以下の5ステップで学習：

1. **学校 → 企業GC+TTPの変換を読む** — 「【学校版】→【企業版 GC + TTP】」の1行で、視点を組織の対話デザインに合わせる
2. **抜き書きを読む** — 原著の該当箇所（重要箇所は **太字**）
3. **実務例を書く** — 自分のGC/TTPの場面に置き換えて言語化
4. **自己評価する** — 4段階ルーブリックで現在地を確認
5. **他アプリで深める** — 関連する3〜5つの他アプリのテーマへリンク

## クロスリンク（5アプリ・エコシステム）

各テーマから、以下の4アプリの関連テーマにリンク：

- **たまかつ**：ありえる楽考の実践知（対話の4モード＝価値共創コミュニティ、との対応強化）
- **限界はあなたの頭の中にしかない**：ジェイ・エイブラハムの思考法
- **コンテキスト・リーダーシップ**：山口周のリーダーシップ論
- **Good to Great 個人キャリア版**：ジム・コリンズの経営研究

対話の4モード（milestone）のみ、たまかつアプリへのリンクを **2件** に増設し、他アプリも含めて計 **5件** の関連リンクを設置。
