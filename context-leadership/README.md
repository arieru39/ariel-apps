# 📖 コンテキスト・リーダーシップ 学習アプリ

山口周『コンテキスト・リーダーシップ』（光文社新書）を素材にした、ありえる楽考スタイルの学習アプリ。

**公開URL**: <https://arieru39.github.io/ariel-apps/context-leadership/>

## 学習観

> 本は「知識を詰める容器」ではなく、「自分の実践を映す鏡」である。
> 読んだあと **「何のために」「誰のために」** の問いに接続されない読書は、まだ読み終わっていない。

## 10テーマ × 3層構造

- **第1層 · 行為論の限界** — `#action_illusion`
- **第2層 · 意味論** — `#meaning_over_action` / `#intent_transmission` / `#delegation_split` / `#feedback_semantics` / `#vision_flex`
- **第3層 · 文脈論** — `#context_reading` / `#interpreter_frame` / `#dialogic_leadership` / `#field_design`

各テーマは 4段階のルーブリック（L1〜L4）で自己評価します。最高段階は「文脈に応じて選び分けられる」で固定 — 本書の主張そのものです。

## 中核ループ（1セッション ≒ 15分）

```
[選ぶ]     今日触る #LO を1つ選ぶ
   ↓
[入る]     本文抜き書き ＋ 問い ＋ 実務例プロンプト
   ↓
[書く]     自分の言葉で100〜300字
   ↓
[写す]     ルーブリック4段階に自己マーク＋根拠1行
   ↓
[つなぐ]   たまかつ接続： 何のために？ 誰のために？
   ↓
[晒す]     場に投げる（Markdown書き出し／保存）
```

## プライバシー

- ✅ 完全にブラウザの中だけで動きます
- ✅ データはあなたのブラウザの localStorage にだけ保存されます
- ✅ サーバーには何も送信されません
- ✅ ログイン不要・インストール不要

## ファイル

- `index.html` — アプリ本体
