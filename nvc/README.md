# NVC 学習アプリ — 実践 → GC発表 → 対話 → レベルアップ版

マーシャル・B・ローゼンバーグ『NVC — 人と人との関係にいのちを吹き込む法 新版』（日本経済新聞出版）を素材に、**NVCを『本で読む知識』から『日常で使える技能』へ育てる** ためのありえる楽考スタイル学習アプリ。

## 誰のためのアプリか

- NVCを学びたいが「読んだだけで終わってしまう」経験を繰り返している人
- グループコーチング（GC）に参加していて、そこで **発表対話** できる練習素材が欲しい人
- 自分・パートナー・親子・チームメンバーとの日常の対話で、**評価・比較・強制を止めたい** と思っている人

## この教材の3つの編集軸

### 1. 「本で読む知識 → 日常で使える技能」への翻訳

各テーマに **`daily_practice`（今日／今週の実践）** を1つ添えました。読んだ直後から、その日のうちに1つだけ実践できる粒度です。

- 【個人版】NVCの概念を読んで理解する → 【GC発表対話版】1週間で1テーマを実践し、GCで発表し、対話でLv1→Lv4にレベルアップする
- 例（テーマ2「観察」）：今日、1つのできごとを『〜だ』ではなく『〜した／〜が起きた』と具体的に書き表す
- 例（テーマ8「共感的傾聴」）：今日、誰かの話に対して『あなたは〜なのね。〜が欲しいのね』と1回言い換えて返す

### 2. 中心マイルストーンは「🌱 誠実さと共感に基づいた絆づくり」

他の10テーマは、この統合状態にたどり着くための構成要素です。**「今、私はどの構成要素でつまずいているか？」の見立てを、GCの発表対話で仲間と共有し、レベルアップの一手を見つけましょう。**

### 3. 「土台 → 4要素の型 → 応用と統合」の3層構造

- **📐 土台（第1層）** — 自分の中の暴力に気づき、自分に共感する
  - いのちを疎外するコミュニケーション／自分への思いやり
- **🧩 4要素の型（第2層）** — 観察／感情／ニーズ／リクエストを実装する
  - 観察 without 評価／感情 vs 思考／感情の責任／普遍的ニーズ／肯定的で具体的なリクエスト
- **🎬 応用と統合（第3層）** — 難しい場面で使いこなす
  - 共感的傾聴／怒りの完全表現／『〜しなければ』を『〜を選ぶ』に／統合実践（絆づくり）

## 11テーマ × 3層構造

### 第1層 · 土台 — 自分の中の暴力に気づき、自分に共感する
1. **📐 『いのちを疎外するコミュニケーション』を見抜くスキル**
2. **🎬 自分への思いやりの基盤** — 完璧でない自分を評価する

### 第2層 · 4要素の型 — 観察／感情／ニーズ／リクエストを実装する
3. **🧩 4要素①：評価を混ぜず『観察』する型**
4. **🧩 4要素②：『感情』と『思考』を区別して表現するスキル**
5. **🧩 4要素③準備：自分の感情に『責任』を持つ型**
6. **🧩 4要素③：普遍的な『ニーズ』を見極める型**
7. **🧩 4要素④：肯定的で具体的な『リクエスト』の型**

### 第3層 · 応用と統合 — 難しい場面で使いこなす
8. **🎬 応用①：共感をもって『聞く』** — 感情とニーズを聞き取る力
9. **🎬 応用②：『怒り』を完全に表現する** — 本当のニーズを暴露する
10. **🎬 応用③：『〜しなければ』を『〜を選ぶ』に翻訳する力**
11. **🌱 統合実践：誠実さと共感に基づいた『絆』づくり** — 中心マイルストーン

## 中核ループ（3ヶ月で全11テーマを一巡する）

各テーマを 1週間で回します：

1. **月：テーマを開く** — 抜き書きを読む＋動画（4分程度）で概観をつかむ
2. **火〜木：日常で実践する** — `daily_practice`（今日／今週の実践）に沿って、日常で1〜3回試す
3. **金〜土：書く** — 実践の中で起きた1場面を、テーマの `prompt`（GC発表用の問い）に沿って100〜300字で記述
4. **GC（週次）：発表対話** — 書いたものをGCで発表し、仲間から『それって思考では？』『本当のニーズは？』などの問い直しを受ける
5. **土日：ルーブリック自己評価** — 4段階のうち、今の自分がどこにいるかを再評価し、次のテーマへ

**3ヶ月で全11テーマを1巡** → 2巡目はLv1→Lv2からLv3→Lv4へ、というふうに、GCの対話がレベルアップの装置になります。

## 想定される使い方（3シナリオ）

### シナリオA：GCに参加している個人
テーマ1（暴力を見抜く）から順に、1週間1テーマで進める。金曜までに `daily_practice` を1〜3回試し、土曜のGCで発表 → 対話 → 自己評価。

### シナリオB：夫婦・親子で並走する
2人で同じテーマを1週間かけて実践し、週末に **お互いにインタビューし合う**（『あなたにとってこのテーマは？』）。GCがなくても、家族が対話相手になる。

### シナリオC：職場のチームで導入
月曜のミーティングで週のテーマを共有 → 各自1週間で実践 → 金曜の30分振り返りで **共感的傾聴のロールプレイ** を1本。3ヶ月で全11テーマを回すと、チームの対話の質が変わる。
