# 掛け書棚 (Kakeshodana) — v0.2.1

**私（鈴木さん）の発想力を鍛えるための、資産棚 × 3つ掛け × 3モード のパーソナルツール。**

- 公開URL: https://arieru39.github.io/ariel-apps/kakeshodana/
- 位置づけ: **個人用の発想筋トレ器具**（コミュニティ配布はまだ）
- スコープ: **1画面・60秒/1ラウンド**

---

## 何をするアプリか

1. **モヤモヤを1行書く**（任意）
2. **資産棚から3つを選ぶ**（本／アプリ／テーマ／動画 から自由に組み合わせ、🎲 ランダムでもOK）
3. **掛け方（モード）を選ぶ**
   - 🌟 **Why掛け** — 3つに共通する目的・あり方・カテゴリー思考を1本抽出する（連結筋）
   - 🧩 **What掛け** — 3つのギャップと補完を三角形で見る（偏見脱筋）
   - 🎬 **How掛け** — 3つの3ステップを1本のプログレッションに繋げる（視座切替筋）
4. **発想を書き出す** → `📋 .md でコピー` してObsidianの `振り返り/発想ログ/` に貼る

---

## 3つの筋肉

| 筋肉 | 手段 | 鍛えられる力 |
|---|---|---|
| **連結筋** | Why掛け | バラバラの資産の中に共通の骨（目的・あり方）を通す |
| **偏見脱筋** | 🎲 ランダム × What掛け | 自分では絶対組まないペアを強制的に見る |
| **視座切替筋** | Why/What/How の切り替え | 同じ3つを、目的・構造・実践の3レイヤーで見直す |

## ファイル構成

```
kakeshodana/
├── index.html      # 1画面UI（縦スクロール完結）
├── registry.json   # 資産インデックス（本・アプリ・テーマ・動画）
└── README.md       # このファイル
```

`registry.json` は `/tmp/build_registry.py` で自動生成される。手動で追記してもよい（次回再生成で消えるが）。

---

## 再生成

```bash
python3 /tmp/build_registry.py
```

Kindle フォルダを増やしたり、ariel-apps に新しいアプリを足したら、これで再生成する。
