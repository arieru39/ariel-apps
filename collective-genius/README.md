## この本の中核主張（1文）

> **集合天才の価値の源泉は「差」だが、集団は放っておくと差を消したがる。**
> 集合天才は「安定状態」ではなく「意図的に維持し続けないと崩れる不安定」である。

---

## 11テーマ × 3層構造

### 第1層 · 差の力学 — なぜ集合天才は難しいのか（心構え）

1. **difficulty_root** — 差が価値／集団は差を消す、逆向き引力の自覚
2. **three_failure_modes** — 3つの安定した失敗形態（同調 / 分裂 / 一人依存）
3. **and_condition_fragility** — 「安全 × 高い基準」の "and" 条件の脆さ

### 第2層 · 場のデザイン — 差を統合するための装置

4. **creative_abrasion** — 創造的摩擦（生産的な対立の設計）
5. **creative_agility** — 創造的機敏さ（速い実験と学習のサイクル）
6. **creative_resolution** — 創造的解決（対立を "and" で保つ第3の道）
7. **psychological_safety_x_standard** — 心理的安全 × 高い基準（Learning Zone）
8. **externalization** — 外化（暗黙知を場に出す動作 · SECI の E）

### 第3層 · 全員主人公化 — response-ability を中心から周縁へ

9. **returning_agency** — 裁量と帰責を本人に返す
10. **shepherd_leadership** — 羊飼い型リーダーシップ（責任の空白を作る）
11. **visible_interdependence** — 相互依存の可視化

## 学習の中核ループ（engine_note）

**Slow で意識化 → 反復で Fast に沈殿 → 文脈で選び分け**

各テーマのルーブリック L1→L4 は、この Fast/Slow の位相を表す：

- L1 気づいていない（Fast が自動で誤作動）
- L2 概念として理解している（Slow で意識すれば使える）
- L3 実践できる（Slow の反復で Fast に沈殿し始めた）
- L4 文脈に応じて選び分けられる（Fast が高解像度／熟達直観）

観点クイズは各選択肢に5層フィードバックを返す：
`stance / phase / phase_note / author_view / slow_question`
