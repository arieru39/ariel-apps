# ariel-apps — ありえる楽考 対話アプリ集

[![GitHub Pages](https://img.shields.io/badge/Live-arieru39.github.io/ariel--apps-2f7d7b)](https://arieru39.github.io/ariel-apps/)

ありえる楽考コミュニティで生まれた、誰でも使える対話アプリ集です。

🔗 **公開URL**: https://arieru39.github.io/ariel-apps/

---

## 🌟 収録アプリ

### 🤲 [助けてください練習アプリ](https://arieru39.github.io/ariel-apps/tasukete.html)

「助けてって言えない」を、4ステップでほぐす練習アプリ。

- **気づく** — ひとりで抱えていることを言語化
- **ほぐす** — 「ありえない」の声を「ありえる」の声に書き換える
- **言葉にする** — 誰に・何を・いつまでに（たまかつ3原則：対等・率直・肯定）
- **口に出す** — コピペで送れる依頼文に

### 🔥 [やれない理由が可能に変わる対話アプリ](https://arieru39.github.io/ariel-apps/yarenai.html)

オントロジカル・コーチング10回コースの **7フェーズ** を1本にまとめた対話アプリ。

- **🧘 ひとりモード／👥 ふたりモード**（コーチ役つき）の2モード
- **60秒の呼吸ガイドアニメ**（4秒吸う／2秒止める／6秒吐く）
- **強い宣言の5条件チェックリスト**
- **音声読み上げ**（ja-JP）で宣言を声に出す
- **Markdownエクスポート** で全プロセスを Obsidian に残せる

### 🪞 [継続する不安が、可能に変わる対話アプリ](https://arieru39.github.io/ariel-apps/fuan.html)

オントロジカル・コーチング **50事例** を鏡として使い、消えない不安を可能に変える対話アプリ。

- **「継続する不安は何ですか？」** から入る7フェーズ構成
- **50事例マッチング**: 不安と口癖から、近い3ケースをサジェスト
- **Before（古い文脈・ラケット）→ Being（在り方の転換）→ Doing（3ステップ）** を実例とともに自分の言葉で書く
- **「どうせ・やっぱり・いつも・絶対」アラート**つき
- **音声読み上げ**＆ **Markdownエクスポート**

### 🔥 [継続する不満が「できるかも」に変わるトレーニング](https://arieru39.github.io/ariel-apps/fuman.html)

ラケット（ペテン）の **4部品 ─ 愚痴／決まった在り方／儲け／代償 ─** を1つずつ自白する **ステップ・バイ・ステップのトレーニングアプリ**。

- **「継続する不満は何ですか？」** から入る10フェーズ構成
- **ラケット4要素**を順番に確定 → 自白 → 代償を直視
- **50事例**を「同じパターンの先輩」として参照しながらBeingをシフト
- **「やれない理由」→「できるかも」言い換え** で柔らかい可能性宣言
- **身体フィーリングチェック**（胸の軽さ／肩の緩み／できるかも度）で頭でなく身体で確かめる
- **ひとりトレーニング／ふたりトレーニングの2モード**（コーチからの問いをそのまま読み上げ可）

## 📦 アプリの特徴

- **完全ローカル動作**: データはブラウザの中だけ。送信なし・保存なし
- **インストール不要**: HTMLファイル1個。ブラウザで開くだけ
- **オフラインOK**: 一度開けば電車の中でも動く
- **シングルファイル**: 各HTMLは依存ゼロのvanilla JS
