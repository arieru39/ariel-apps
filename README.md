## 🌟 収録アプリ

### 🤲 [助けてください練習アプリ](https://arieru39.github.io/ariel-apps/tasukete.html)

「助けてって言えない」を、4ステップでほぐす練習アプリ。

- **気づく** — ひとりで抱えていることを言語化
- **ほぐす** — 「ありえない」の声を「ありえる」の声に書き換える
- **言葉にする** — 誰に・何を・いつまでに（たまかつ3原則：対等・率直・肯定）
- **口に出す** — コピペで送れる依頼文に

### 🔥 [やれない理由が可能に変わる対話アプリ](https://arieru39.github.io/ariel-apps/yarenai.html)

オントロジカル・コーチング10回コースの **7フェーズ** を1本にまとめた対話アプリ。

- **🧘 ひとりモード／👥 ふたりモード**（コーチ役つき）の2モード
- **60秒の呼吸ガイドアニメ**（4秒吸う／2秒止める／6秒吐く）
- **強い宣言の5条件チェックリスト**
- **音声読み上げ**（ja-JP）で宣言を声に出す
- **Markdownエクスポート** で全プロセスを Obsidian に残せる

### 🪞 [継続する不安が、可能に変わる対話アプリ](https://arieru39.github.io/ariel-apps/fuan.html)

オントロジカル・コーチング **50事例** を鏡として使い、消えない不安を可能に変える対話アプリ。

- **「継続する不安は何ですか？」** から入る7フェーズ構成
- **50事例マッチング**: 不安と口癖から、近い3ケースをサジェスト
- **Before（古い文脈・ラケット）→ Being（在り方の転換）→ Doing（3ステップ）** を実例とともに自分の言葉で書く
- **「どうせ・やっぱり・いつも・絶対」アラート**つき
- **音声読み上げ**＆ **Markdownエクスポート**

### 🔥 [継続する不満が「できるかも」に変わるトレーニング](https://arieru39.github.io/ariel-apps/fuman.html)

「ずっと言い続けている不満」を入口に、**7つの問い** でラケットを暴き、**古いあり方 → 新しいあり方** のシフトを言葉と身体で体感するトレーニングアプリ。

- **問い①〜④**：継続する不満 → 古いあり方 → 儲け → 代償（ラケット4要素を順番に自白）
- **問い⑤（核心）**：A「手放す」／ B「続ける」── どちらを選ぶかを自分で決める
- **問い⑥**：「私は（誰）が（　　）になる場です」の文法テンプレートで **新しいあり方** を言語化
- **問い⑦**：新しいあり方から開く **ビジョン** を言葉で宣言
- **古いあり方 → 新しいあり方** のシフトを画面上に対比表示（3画面で繰り返し可視化）
- 各問いに **3〜6個の言い換えバリエーション** を用意（ふたりトレーニング向け）
- **50事例マッチング**を「同じパターンの先輩」として参照しながらシフト
- **身体フィーリングチェック**（胸の軽さ／肩の緩み／できるかも度）で頭でなく身体で確かめる
- **ひとりトレーニング／ふたりトレーニングの2モード**

### 🔭 [原理レベル思考トレーニング — 達人のように考えるための修練](https://arieru39.github.io/ariel-apps/genri-thinking.html)

Anders Ericsson『超一流になるのは才能か努力か？』**最終章** と、カール・ワイマンの「奇跡の授業」（41% vs 74%）を骨に、**達人のあり方の最高峰** を目指す修練アプリ。ストレス時に反射的に出る「強い手札」とは**別階層**のトレーニング。

- **7フェーズ構成**：START → 判別ゲーム → 3層分解 → 転移テスト → 自分の仕事 → 達人のあり方の宣言 → サマリー
- **判別ゲーム（6問）**：営業／育成／新規事業／ダイエット／会議／子育て。各場面で「知識レベルの答え」と「原理レベルの答え」を判別、即フィードバック付き
- **3層分解トレーニング**：「なぜスタバでコーヒーがあの値段で売れるのか？」を **表面 → パターン → 原理（サードプレイス）** に降ろす
- **転移テスト**：掴んだ原理を **銭湯／ジム／ありえる楽考** の別ドメインに転移。Mental Representation ネットワークの可視化つき
- **自分の仕事に持ち帰る**：職業・現象を選んで3層分解＋2つの遠いドメインへ転移
- **達人のあり方の宣言**：修練で Mental Representation を精緻にし、**これまで聞こえなかったコンテキストが聞こえる自分になる** ── ホモ・エクセサンス（練習する人）としてのBeing宣言、音声読み上げ付き
- **接続する実践知**：センスメイキング（カール・ワイク）／コンテキスト・リーダーシップ（石井遼介）／Mental Representation（Ericsson）── これらの共通土台としての「原理レベルで考える」筋トレ
- **たまかつとの構造的同型**：現象 → モデル → コンセプト の3層構造を最後に接続
- **達人の実践知 系統・第1弾**（強い手札とは階層が違う、最高峰のあり方）

### 🌱 [共創関係イメージング — 相手の成功が自分の利益になる50事例](https://arieru39.github.io/ariel-apps/co-creation.html)

「自分の力だけで頑張る」ゲームから、**「相手の成功が、自分の利益の源になる」** ゲームへの視点転換を、**8カテゴリ×50事例** で炙り出すアプリ。

- **50事例ライブラリ**：サプライチェーン(10)／クリエイティブ(6)／プロフェッショナル(8)／育成・教育(8)／投資・金融(4)／プラットフォーム(6)／アライアンス(4)／地域・コミュニティ(4)
- **各事例5要素構造**：役割 ／ 相手 ／ 連鎖ストーリー ／ 気づき ／ 明日できる最初の1歩
- **カテゴリ絞り込み & キーワード検索**、⭐でピック（参考事例として保存）
- **自分の共創マップ**：役割・相手・連鎖ストーリーを書く。参考事例が横に並ぶ
- **共創宣言 & 24時間以内の1歩**：宣言文自動生成 ／ 音声読み上げ（ja-JP） ／ Markdownエクスポート
- 「強い手札」系統の第1弾（ラケット系と対をなす）

### 🌍 [ピースデー地球一周リレー 仲間集めアプリ](https://arieru39.github.io/ariel-apps/peace-relay.html)

9月21日（世界平和の日）に **24時間で地球を一周するリレー** を、**無理なく** 実現するための仲間集めアプリ。

- **「24時間ぜんぶ集める」を捨てて、見える化に置き換える** 設計
- **53都市の簡易タイムゾーン辞書**（東アジア〜オセアニアまで地域別 optgroup）＋手入力対応
- **24時間タイムライン可視化** ── UTC基準で誰がどこにいるか、空き時間は赤マス
- **空き時間分析** ── その時間が朝〜夕方になっている地域＋都市例を自動ヒント表示
- **招待文テンプレ 3バージョン** ── 日本語／英語／"友人の友人を紹介して" 依頼
- **Markdownエクスポート＆シェア用テキスト** で進捗を残せる・広げられる
- **完全ローカル動作** ── 友人リストは送信なし、ブラウザの中だけ

## 📦 アプリの特徴

- **完全ローカル動作**: データはブラウザの中だけ。送信なし・保存なし
- **インストール不要**: HTMLファイル1個。ブラウザで開くだけ
- **オフラインOK**: 一度開けば電車の中でも動く
- **シングルファイル**: 各HTMLは依存ゼロのvanilla JS
