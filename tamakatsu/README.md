# 🌱 価値共創コミュニティで「たまかつ」を生きる 学習アプリ

ありえる楽考『価値共創コミュニティで「たまかつ」を生きる — Kindle版』を素材にした、ありえる楽考スタイルの学習アプリ。

**公開URL**: <https://arieru39.github.io/ariel-apps/tamakatsu/>

## この本の主張（1文）

**「平凡な個人が無意識の思い込みを修正し、自らの『たまかつ』を見出し、グループコーチングと経験学習を通じて熟達に至ることで、価値共創コミュニティで貢献志向の生き方を実現できる」**

「ありえない」の景色を「ありえる」の未来に変える方法を、認知強化（思考のエラーを媒介する）→ 熟達（心的表象で世界の解像度を上げる）→ 価値共創（場をみんなでつくる）の順で辿り、最終的に **自分軸と相手軸の交差点＝たまかつ** に立つ。

## 12テーマ × 3層構造

- **第1層 · 認知強化** — `#unconscious_assumption` / `#four_cognitive_strategies` / `#symbol_grounding`
- **第2層 · 熟達と場づくり** — `#expertise_resolution` / `#bottleneck_concentration` / `#group_coaching` / `#three_layer_team`
- **第3層 · 価値共創** — `#tamakatsu_calling` / `#collective_space` / `#experience_to_product` / `#recruitment_vision` / `#declaration_system`

層は順番に上るのではなく、螺旋的に往復する。思考が変わる → 熟達の解像度が上がる → 場が育つ → 自分の "たまかつ" が動き出す → 仲間が集まる → さらに深い思考が生まれる。

## 中核ループ

```
[選ぶ]     今日触る #LO を1つ選ぶ
   ↓
[入る]     本文抜き書き ＋ 種の問い ＋ 実務例プロンプト
   ↓
[書く]     自分の言葉で100〜300字
   ↓
[写す]     ルーブリック4段階に自己マーク＋根拠1行
   ↓
[つなぐ]   たまかつ接続： 誰のために？ 何のために？
   ↓
[晒す]     場に投げる（Markdown書き出し／保存）
```
