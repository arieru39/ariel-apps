# 🌱 価値共創コミュニティで「たまかつ」を生きる 学習アプリ

ありえる楽考『価値共創コミュニティで「たまかつ」を生きる — Kindle版』を素材にした、ありえる楽考スタイルの学習アプリ。

**公開URL**: <https://arieru39.github.io/ariel-apps/tamakatsu/>

## この本の主張（1文）

**「平凡な個人が無意識の思い込みを修正し、自らの『たまかつ』を見出し、グループコーチングと経験学習を通じて熟達に至ることで、価値共創コミュニティで貢献志向の生き方を実現できる」**

「ありえない」の景色を「ありえる」の未来に変える方法を、認知強化（思考のエラーを媒介する）→ 熟達（心的表象で世界の解像度を上げる）→ 価値共創（場をみんなでつくる）の順で辿り、最終的に **自分軸と相手軸の交差点＝たまかつ** に立つ。

## 12テーマ × 3層構造

- **第1層 · 認知強化** — `#unconscious_assumption` / `#four_cognitive_strategies` / `#symbol_grounding`
- **第2層 · 熟達と場づくり** — `#expertise_resolution` / `#bottleneck_concentration` / `#group_coaching` / `#three_layer_team`
- **第3層 · 価値共創** — `#tamakatsu_calling` / `#collective_space` / `#experience_to_product` / `#recruitment_vision` / `#declaration_system`

層は順番に上るのではなく、螺旋的に往復する。思考が変わる → 熟達の解像度が上がる → 場が育つ → 自分の "たまかつ" が動き出す → 仲間が集まる → さらに深い思考が生まれる。

## 中核ループ

```
[入口①]   🎯 関係性クイズ（4択・8問）    ← NEW: 書く前に主要概念の関係を掴む
[入口②]   ✍️ 第1層 #unconscious_assumption から書く    ← NEW: 第1層ガイド
   ↓
[選ぶ]     今日触る #LO を1つ選ぶ（第1層START印が目印）
   ↓
[入る]     本文抜き書き ＋ 種の問い ＋ 実務例プロンプト
   ↓
[書く]     自分の言葉で100〜300字
   ↓
[写す]     ルーブリック4段階に自己マーク＋根拠1行（⚡ Fast/Slow の位相付き）
   ↓
[つなぐ]   たまかつ接続： 誰のために？ 何のために？
   ↓
[晒す]     場に投げる（Markdown書き出し／保存）
```

## 学習エンジン（原理レベル思考トレーニングから移植）

このアプリの背骨は、姉妹アプリ [原理レベル思考トレーニング](../genri-thinking.html) と同じ **達人の実践知エンジン** です：

- **判別ゲーム（4択）** — 「関係性クイズ」として実装。書くハードルの前に、選ぶだけで主要概念の関係が掴める
- **層構造ガイド** — 第1層（土台）から始める順序を明示。START印で入口を可視化
- **Fast / Slow ルーブリック** — L1〜L4を、Kahneman の System 1（Fast）と System 2（Slow）の位相として読み直し。「Slow で意識化 → 反復で Fast に沈殿 → 文脈で選び分け」の熟達曲線
- **統一原理** — 「**Slow で意図的に検証・失敗・修正した経験が、Fast に沈殿する**」（Ericsson『超一流になるのは才能か努力か?』最終章／Kahneman『Fast & Slow』／Gladwell『第1感』／Klein × Kahneman 2009 熟達直観）

## 4択関係性クイズ（8問）

いきなり書くのが重かったら、まず4択で全体像を掴めます：

1. 12テーマの3層構造 — なぜ第1層から始めるか
2. 第1層と第3層の関係 — 思い込みとたまかつ
3. 記号接地とは何か — 「わかる」の正体
4. 熟達の本質 — 心的表象と解像度
5. ボトルネック集中 — 制約理論の核心
6. グループコーチング — 「聞かれる」が答えを引き出す
7. 三層構造 — なぜ／何が必要か／どうやって
8. 12テーマの中心 — たまかつを生きる

即フィードバック付き。触った主要テーマが最後に一覧で表示されます。
